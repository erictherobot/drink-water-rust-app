# RUST Drink Water Reminder

Not much here. just a simple water reminder app that helps me drink more water.

## How to use

1. Clone the repo
2. Run `cargo run` in the root directory
3. Enjoy!

## How to build

1. Clone the repo
2. Run `cargo build --release` in the root directory
3. Enjoy!
